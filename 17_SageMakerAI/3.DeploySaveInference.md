# Deploy and Save Model for Inference using AWS SageMaker

### Create Inference Script (UI)

Inside SageMaker Studio:

Click File → New → Text File

Name it: inference.py

Paste this:

```python
import joblib
import os
import json

def model_fn(model_dir):
    return joblib.load(os.path.join(model_dir, "iris-model.pkl"))

def input_fn(request_body, request_content_type):
    data = json.loads(request_body)
    return data["instances"]

def predict_fn(input_data, model):
    return model.predict(input_data)

def output_fn(prediction, accept):
    return json.dumps(
        {"predictions": prediction.tolist()}
    ), "application/json"
```

1. `import joblib, os, json` → Imports libraries for loading model, file paths, and JSON handling.
2. `model_fn(model_dir)` → SageMaker calls this to load the trained model when endpoint starts.
3. `joblib.load(...)` → Loads `iris-model.pkl` file from model directory.
4. `input_fn(request_body, request_content_type)` → Handles incoming API request data.
5. `json.loads(request_body)` → Converts JSON request body into Python dictionary.
6. `return data["instances"]` → Extracts input records for prediction.
7. `predict_fn(input_data, model)` → Runs prediction using loaded model.
8. `model.predict(input_data)` → Uses ML model to predict Iris flower class.
9. `output_fn(prediction, content_type)` → Formats prediction response back to client.
10. `json.dumps({"predictions": prediction.tolist()})` → Returns predictions as JSON. ([sagemaker.readthedocs.io][1])

[1]: https://sagemaker.readthedocs.io/en/v2.29.2/frameworks/sklearn/using_sklearn.html "Using Scikit-learn with the SageMaker Python SDK"

Save the file.

### Package Model for SageMaker

SageMaker does NOT deploy loose files. It expects a tar.gz file.

Run this in the notebook:

```bash
import tarfile

with tarfile.open("model.tar.gz", "w:gz") as tar:
    tar.add("iris-model.pkl")
    tar.add("inference.py")

print("model.tar.gz created")
```

### Upload model.tar.gz to S3

```bash
import boto3

s3 = boto3.client("s3")
bucket_name = "my-sagemaker-demo-bucket-temp"
s3.upload_file(
    "model.tar.gz",
    bucket_name,
    "model-artifacts/model.tar.gz"
)

print("Packaged model uploaded to S3")
```

Now SageMaker can deploy this model.

### Optional python code

```python
import warnings
warnings.filterwarnings("ignore", category=SyntaxWarning)
from sagemaker.sklearn.model import SKLearnModel

model = SKLearnModel(
    model_data='<s3 model path>',
    role='<your-sagemaker-execution-role>',
    entry_point='inference.py',
    framework_version='1.4-2'   # Use version matching your local environment
)

predictor = model.deploy(
    instance_type='ml.t2.medium',
    initial_instance_count=1
)
```
Note: you might get syntax warnings, but this is due to we are usign python 3.12 and not 3.11, either use python 3.11 or ignore these warnings

### Create Endpoint Configuration (UI)

Go to Inference → Endpoint configurations

Click Create endpoint configuration

Name: `iris-endpoint-config`

Production variant

Model: `iris-demo-model`

Instance type: `ml.t2.medium`

Initial instance count: `1`

Click Create

### Create Endpoint 

Go to Inference → Endpoints

Click Create endpoint

Endpoint name: `iris-demo-endpoint`

Select endpoint configuration: `iris-endpoint-config`

Click Create endpoint

⏳ Wait ~5–7 minutes

Status should become InService
