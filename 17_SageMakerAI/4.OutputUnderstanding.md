## Meaning of These Inputs

Our model uses the **Iris flower dataset**.
Each inner list has **4 flower measurements (in cm)** in this exact order:

```text
[Sepal Length, Sepal Width, Petal Length, Petal Width]
```

Scikit-learn Iris dataset uses these 4 features and predicts one of 3 species.

---

# Output Labels Meaning

```text
0 = Setosa
1 = Versicolor
2 = Virginica
```

---

# 1️⃣ Input

```json
{
  "instances": [[5.1, 3.5, 1.4, 0.2]]
}
```

## Means

| Feature      | Value  |
| ------------ | ------ |
| Sepal Length | 5.1 cm |
| Sepal Width  | 3.5 cm |
| Petal Length | 1.4 cm |
| Petal Width  | 0.2 cm |

## Expected Outcome

```json
{"predictions":[0]}
```

👉 **Setosa** (small petals are typical setosa)

# 3️⃣ Batch Input

```json
{
  "instances": [
    [5.1,3.5,1.4,0.2],
    [6.7,3.1,4.7,1.5],
    [7.2,3.6,6.1,2.5]
  ]
}
```

## Means 3 flowers sent together

| Row               | Likely Species | Output |
| ----------------- | -------------- | ------ |
| [5.1,3.5,1.4,0.2] | Setosa         | 0      |
| [6.7,3.1,4.7,1.5] | Versicolor     | 1      |
| [7.2,3.6,6.1,2.5] | Virginica      | 2      |

## Expected Outcome

```json
{"predictions":[0,1,2]}
```
